# Codebase Summary

Agent-Squad is a multi-agent AI system with 60K+ Python backend LOC and 9K+ TypeScript frontend LOC.

## Backend (60,072 LOC)
- Agent System: 22,912 LOC across 79 files
- API Layer: 8,928 LOC across 32 endpoints  
- Clean architecture with FastAPI
- Agno framework for agent orchestration

## Frontend (9,619 LOC)
- Next.js 16 with React 19
- 47 components with shadcn/ui
- Real-time Kanban and chat features

## Key Features
- 12 specialized agent roles
- Discovery-driven workflows
- Real-time collaboration
- Production-ready architecture

Technology: FastAPI, PostgreSQL, NATS, Next.js, Tailwind CSS
